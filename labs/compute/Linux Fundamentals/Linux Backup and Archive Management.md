# Linux Backup and Archive Management

## Overview

This lab focused on creating, managing, and transferring backups within a Linux environment running on Amazon EC2. The exercises introduced archive creation using the `tar` utility, backup logging, and file movement between departments to simulate real-world backup and data management workflows.

The lab demonstrated how Linux administrators can package entire directory structures into compressed archives, maintain backup records, and securely transfer backup files between teams.

---

**Task 1: Create a Compressed Backup Archive**

Created a complete backup of a company directory structure using Linux archiving tools.

Key activities completed:

* Verified an existing multi-level directory structure.
* Used the `tar` utility to create a compressed archive.
* Included all folders and files recursively within the backup.
* Validated successful archive creation.

Skills demonstrated:

* Linux backup management
* Archive creation using `tar`
* Recursive file processing
* File system verification

---

**Task 2: Maintain a Backup Log**

Created and maintained a backup log to track backup activity.

Key activities completed:

* Created a backup tracking file.
* Recorded backup date, time, and archive filename.
* Used Linux output redirection techniques to write information to a file.
* Verified backup log contents.

Skills demonstrated:

* Logging and documentation
* Output redirection
* File content management
* Linux administration best practices

---

**Task 3: Transfer Backup Files**

Moved backup archives between organizational folders to simulate data sharing and recovery workflows.

Key activities completed:

* Transferred a backup archive to another department directory.
* Verified successful file relocation.
* Confirmed archive availability in the destination folder.

Skills demonstrated:

* File movement and relocation
* Data organization
* Backup distribution workflows
* Linux file management

---

**Key Learning Outcomes**

Completed practical exercises covering:

* Linux backup creation and management.
* Archive compression using tar.
* Backup record keeping and auditing.
* File transfer and relocation.
* Administrative workflows commonly used in cloud and enterprise environments.
