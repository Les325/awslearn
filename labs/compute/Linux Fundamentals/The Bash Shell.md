# The Bash Shell

## Overview

This lab focused on customizing and improving the Linux command-line environment using the Bash shell. The exercises introduced command aliases, environment variables, and executable search paths to simplify repetitive administrative tasks and improve command-line efficiency.

The lab demonstrated how Bash can be configured to automate common operations, streamline workflows, and manage executable programs within a Linux environment running on Amazon EC2.

---

**Task 1: Create a Backup Alias**

Created a custom Bash alias to simplify the process of backing up directories using the `tar` utility.

Key activities completed:

* Created a reusable `backup` alias.
* Configured the alias to generate compressed archive files.
* Executed the alias to back up a complete company directory.
* Verified successful archive creation.

Skills demonstrated:

* Bash alias creation
* Linux command customization
* Archive creation using `tar`
* Backup automation

---

**Task 2: Explore the PATH Environment Variable**

Examined how Linux locates executable programs using the `PATH` environment variable.

Key activities completed:

* Executed shell scripts from different directory locations.
* Investigated why commands could not be executed directly.
* Displayed the current `PATH` environment variable.
* Added a custom directory to the `PATH`.
* Successfully executed scripts without specifying their full path.

Skills demonstrated:

* Environment variable management
* PATH configuration
* Executable search paths
* Linux shell navigation

---

**Task 3: Execute Shell Scripts**

Worked with executable shell scripts stored in custom directories.

Key activities completed:

* Executed scripts using relative paths.
* Executed scripts using direct file paths.
* Configured the shell to locate executables automatically.
* Verified successful script execution after updating the environment.

Skills demonstrated:

* Bash scripting fundamentals
* Script execution
* Relative and absolute paths
* Linux command execution

---

**Key Learning Outcomes**

Completed practical exercises covering:

* Creating and using Bash aliases.
* Automating repetitive administrative tasks.
* Managing Linux environment variables.
* Configuring the `PATH` variable.
* Executing shell scripts from custom directories.
* Understanding how the Bash shell locates executable programs.
