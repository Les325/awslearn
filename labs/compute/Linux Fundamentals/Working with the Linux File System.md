# Working with the Linux File System

## Overview

This lab focused on Linux file system management and directory organization within an Amazon Linux environment running on Amazon EC2. The exercises covered creating directory structures, creating and managing files, navigating the Linux file system, and reorganizing content using common Linux commands.

The lab simulated a real-world business scenario where company departments and records needed to be organized and later restructured to meet changing business requirements.

---

**Task 1: Create a Multi-Level Directory Structure**

Created a company file structure using Linux command-line tools.

Key activities completed:

* Created a top-level company directory.
* Created departmental subdirectories for Finance, HR, and Management.
* Created multiple CSV files within each department.
* Navigated the directory structure using relative and absolute paths.
* Validated folder and file creation using Linux file listing commands.

Skills demonstrated:

* Directory creation (`mkdir`)
* File creation (`touch`)
* Linux path navigation (`cd`)
* Directory verification (`ls`, `pwd`)
* Understanding hierarchical file systems

---

**Task 2: Manage and Verify Files and Directories**

Used Linux commands to inspect and validate file system contents.

Key activities completed:

* Listed files and directories at multiple levels.
* Verified current working directory locations.
* Used recursive listing commands to inspect complete directory structures.
* Practiced working with relative paths and parent directories.

Skills demonstrated:

* File system inspection
* Recursive directory analysis
* Relative and absolute path management
* Linux command-line navigation

---

**Task 3: Reorganize Company Data**

Modified an existing directory structure to meet new organizational requirements.

Key activities completed:

* Copied directories and their contents using recursive copy operations.
* Moved folders into new locations within the hierarchy.
* Created new directories and relocated files.
* Deleted files and removed unused directories.
* Verified changes after each operation.

Skills demonstrated:

* Copying directories (`cp -r`)
* Moving files and folders (`mv`)
* Deleting files (`rm`)
* Removing directories (`rmdir`)
* File system restructuring and administration

---

**Key Learning Outcomes**

Completed practical exercises covering:

* Linux file system organization.
* File and directory management.
* Navigation using relative and absolute paths.
* Data migration and folder restructuring.
* Core Linux administration skills commonly used in cloud and DevOps environments.
