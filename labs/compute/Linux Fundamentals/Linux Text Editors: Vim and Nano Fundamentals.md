# Linux Text Editors: Vim and Nano Fundamentals

## Overview

This lab focused on developing practical Linux command-line skills by working with two of the most commonly used text editors in Linux environments: Vim and Nano. The exercises covered file creation, editing, saving, and navigation techniques that are frequently used when administering Linux servers in cloud environments such as Amazon EC2.

---

**Task 1: Complete the Vim Tutorial**

Used the built-in `vimtutor` application to learn the fundamentals of the Vim text editor.

Key activities completed:

* Launched the Vim tutorial environment using `vimtutor`.
* Completed introductory lessons covering navigation and editing techniques.
* Learned the different Vim modes, including command mode and insert mode.
* Practiced cursor movement, text editing, and exiting Vim safely.

Skills demonstrated:

* Vim editor fundamentals
* Linux terminal navigation
* Command-mode text editing
* Keyboard-driven workflow efficiency

---

**Task 2: Create and Edit Files Using Vim**

Created and modified text files using the Vim editor.

Key activities completed:

* Created a new file named `helloworld`.
* Entered insert mode to add text content.
* Saved changes using Vim write commands.
* Reopened and modified an existing file.
* Demonstrated the difference between saving changes (`:wq`) and exiting without saving (`:q!`).

Additional commands explored:

* `dd` – delete a line
* `u` – undo changes
* `:w` – save without exiting

Skills demonstrated:

* File creation and management
* Editing text in Vim
* Saving and discarding changes
* Basic Linux file manipulation

---

**Task 3: Create and Edit Files Using Nano**

Used Nano as an alternative Linux text editor and compared its workflow to Vim.

Key activities completed:

* Created a file named `cloudworld`.
* Entered and modified text directly without using editing modes.
* Saved files using keyboard shortcuts.
* Reopened files to verify saved content.

Skills demonstrated:

* Nano text editor usage
* File editing and persistence
* Linux command-line productivity
* Understanding differences between text editors

---

**Key Learning Outcomes**

Completed hands-on exercises covering:

* Linux text editor fundamentals.
* File creation, editing, and management from the command line.
* Differences between Vim and Nano workflows.
* Saving, modifying, and validating file contents.
* Essential Linux skills used in server administration, cloud engineering, and DevOps environments.
